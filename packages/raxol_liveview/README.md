# Raxol LiveView

Phoenix LiveView integration for Raxol terminals.

**Status: Scaffold -- not yet wired.** This package exists as a placeholder for the LiveView bridge extraction. The working LiveView integration currently lives in the main `raxol` package at `lib/raxol/live_view/`.

See [LiveView cookbook](../../docs/cookbook/LIVEVIEW_INTEGRATION.md) for usage patterns.
