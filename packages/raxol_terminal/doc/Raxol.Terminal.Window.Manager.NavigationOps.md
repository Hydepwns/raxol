# `Raxol.Terminal.Window.Manager.NavigationOps`
[🔗](https://github.com/DROOdotFOO/raxol/blob/v2.4.0/lib/raxol/terminal/window/manager/navigation_ops.ex#L1)

Pure-functional helpers for spatial navigation and position registration
within the WindowManagerServer state.

# `define_navigation_path`

Defines a navigation path between windows using NavigationUtils.

# `register_window_position`

Registers a window's spatial position data in the spatial_map.
