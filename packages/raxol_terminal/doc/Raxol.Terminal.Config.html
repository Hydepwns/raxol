    Raxol.Terminal.Config (Raxol Terminal v2.4.0)

    Copy Markdown View Source

    Handles terminal settings and behavior, including:

    • Terminal dimensions
    • Color settings
    • Input handling
    • Terminal state management
    • Configuration validation
    • Configuration persistence
