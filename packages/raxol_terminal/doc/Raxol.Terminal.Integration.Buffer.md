# `Raxol.Terminal.Integration.Buffer`
[🔗](https://github.com/DROOdotFOO/raxol/blob/v2.4.0/lib/raxol/terminal/integration/integration_buffer.ex#L1)

Handles buffer and cursor management for the terminal.

# `clear`

Clears the terminal buffer.

# `get_cursor_position`

Gets the current cursor position.

# `get_scroll_position`

Gets the current scroll position.

# `get_total_lines`

Gets the total number of lines in the buffer.

# `get_visible_content`

Gets the current visible content.

# `get_visible_lines`

Gets the number of visible lines.

# `move_cursor`

Moves the cursor to a specific position.

# `resize`

Resizes the terminal buffer.

# `scroll`

Scrolls the terminal buffer.

# `write`

Writes text to the terminal buffer.
