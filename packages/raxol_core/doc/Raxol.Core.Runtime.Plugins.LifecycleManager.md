# `Raxol.Core.Runtime.Plugins.LifecycleManager`
[🔗](https://github.com/DROOdotFOO/raxol/blob/v2.4.0/lib/raxol/core/runtime/plugins/lifecycle_manager.ex#L1)

Handles plugin lifecycle operations including loading, unloading, enabling, and disabling plugins.

# `cleanup_plugin`

Cleans up a plugin.

# `disable_plugin`

Disables a plugin.

# `enable_plugin`

Enables a plugin.

# `handle_event`

Handles an event.

# `initialize_plugin`

Initializes a plugin with the given configuration.

# `initialize_plugins`

Initializes all plugins.

# `load_plugin`

Loads a plugin with the given configuration.

# `load_plugin_by_module`

Loads a plugin by module.

# `reload_plugin`

Reloads a plugin.

# `reload_plugin_from_disk`

Reloads a plugin from disk.

# `unload_plugin`

Unloads a plugin.
