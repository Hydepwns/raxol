    Raxol.Core.FocusManager (Raxol Core v2.4.0)

    Copy Markdown View Source

    Refactored FocusManager that delegates to GenServer implementation.

    This module provides the same API as the original FocusManager but uses a supervised GenServer instead of the Process dictionary for state management.

    Migration Notice

    This module is a drop-in replacement for Raxol.Core.FocusManager. All functions maintain backward compatibility while providing improved fault tolerance and functional programming patterns.

    Benefits over Process Dictionary

    • Supervised state management with fault tolerance
    • Pure functional transformations
    • Better debugging and testing capabilities
    • Clear separation of concerns
    • No global state pollution
