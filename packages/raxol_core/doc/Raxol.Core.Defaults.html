    Raxol.Core.Defaults (Raxol Core v2.4.0)

    Copy Markdown View Source

    Canonical default values for the Raxol framework.

    These compile-time constants eliminate magic numbers across packages. All packages depend on raxol_core, so these are universally accessible.
