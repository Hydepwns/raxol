# `Raxol.Core.Accessibility.PreferenceManager`
[🔗](https://github.com/DROOdotFOO/raxol/blob/v2.4.0/lib/raxol/core/accessibility/preference_manager.ex#L1)

Preference sync helpers for AccessibilityServer.
Handles merging, setting, and syncing preferences to UserPreferences.

# `maybe_notify_color_system`

Notifies ColorSystem.Server when high_contrast preference changes.

# `merge`

Merges option keyword list into the current preferences map.

# `sync_all`

Syncs all accessibility preferences to the UserPreferences process.

# `sync_one`

Syncs a single preference key/value to the UserPreferences process.
