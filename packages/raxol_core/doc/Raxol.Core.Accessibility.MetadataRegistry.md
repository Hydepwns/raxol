# `Raxol.Core.Accessibility.MetadataRegistry`
[🔗](https://github.com/DROOdotFOO/raxol/blob/v2.4.0/lib/raxol/core/accessibility/metadata_registry.ex#L1)

Pure-functional helpers for accessibility metadata and component style
registration within the AccessibilityServer state.

# `get_component_style`

Gets a component style keyed by `{:component_style, type}`.

# `get_metadata`

Gets element/component metadata from state.

# `put_metadata`

Puts element/component metadata into state.

# `register_component_style`

Registers a component style keyed by `{:component_style, type}`.

# `remove_metadata`

Removes element/component metadata from state.

# `unregister_component_style`

Unregisters a component style.
