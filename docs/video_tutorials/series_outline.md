# Raxol Video Tutorial Series

10 episodes covering terminal UI development with Raxol.

---

## Episode 1: First Terminal App (45 min, beginner)

Setup, hello world, TEA model basics (`init/update/view`), terminal coordinate system.

Demo: Counter app with keyboard input and button clicks.

## Episode 2: Components (50 min, beginner-intermediate)

Component lifecycle, props vs state, composition patterns. Building a button, text input, toggle, and keyboard shortcuts.

Demo: Calculator.

## Episode 3: Layout and Styling (40 min, intermediate)

Flexbox/grid layouts, ANSI colors, theming, light/dark modes, border styles, responsive sizing.

Demo: Multi-panel dashboard with theme switching.

## Episode 4: Data and Lists (45 min, intermediate)

Dynamic data, scrollable lists with virtual scrolling, sortable/filterable tables, pagination.

Demo: File browser with search.

## Episode 5: Forms and Input (40 min, intermediate)

Form state, validation, error display. Text inputs, select dropdowns, autocomplete, multi-step wizards.

Demo: Registration form with validation.

## Episode 6: Real-time Updates (45 min, advanced)

LiveView integration, PubSub, live data visualization, throttling/debouncing.

Demo: Live system monitor with CPU/memory graphs.

## Episode 7: Navigation (40 min, intermediate-advanced)

Multi-screen apps, modals, breadcrumbs, back/forward history, state persistence across screens.

Demo: Terminal email client.

## Episode 8: Testing and Debugging (35 min, intermediate-advanced)

Unit testing components, integration testing workflows, time-travel debugger, performance profiling.

Demo: Adding tests to previous examples.

## Episode 9: Performance (50 min, advanced)

Virtual DOM diffing, batch rendering, memory management, production profiling.

Demo: Optimizing a complex dashboard.

## Episode 10: Production (60 min, advanced)

Plugin system, SSH deployment, Fly.io, Docker, monitoring, accessibility.

Demo: Deploying an app with SSH access.

---

## Prerequisites

- Basic Elixir knowledge
- Terminal familiarity
- Elixir 1.15+, terminal with ANSI support
