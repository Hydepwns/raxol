# Core Examples

Basic usage of Raxol's core modules. For a complete interactive catalog of all widgets and charts, run:

```bash
mix raxol.playground
```

## Running Examples

```bash
# Minimal counter (TEA pattern)
mix run examples/getting_started/counter.exs

# Todo list
mix run examples/apps/todo_app.ex

# File browser with tree nav
mix run examples/apps/file_browser.exs

# Flagship demo (live BEAM dashboard)
mix run examples/demo.exs

# Built-in demo runner
mix raxol.demo
```

See the [Quickstart](../../docs/getting-started/QUICKSTART.md) for a walkthrough of the TEA model.
